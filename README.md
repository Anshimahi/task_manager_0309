# 📱 Task Manager App (Flutter)

A simple and clean Flutter application to manage daily Tasks, Schedule, and Notes.

---

## ✨ Features

* ✅ Add tasks with date & time picker
* ✅ Mark tasks as completed
* ✅ Delete tasks
* 📅 Schedule view:

  * Today tasks
  * Upcoming tasks
* 📝 Notes section (Grid layout)

---

## 🛠 Tech Stack

* Flutter
* Dart

---

## 🚀 How to Run

```bash
flutter pub get
flutter run
```
